# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- Initial release of mcp-server-kubernetes.
- Tool: `list_pods` - List pods in a namespace.
- Tool: `list_deployments` - List deployments in a namespace.
- Tool: `list_services` - List services in a namespace.
- Tool: `get_pod_logs` - Get logs from a pod.
- Tool: `apply_manifest` - Apply a Kubernetes manifest.
- Tool: `list_namespaces` - List all namespaces.
- Tool: `get_events` - Get cluster events.
- Tool: `scale_deployment` - Scale a deployment replica count.
- MCP server with stdio transport.
- Docker support.
